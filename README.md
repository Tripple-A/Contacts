## To Start
```
Add sudo if you need permissions :)
docker-compose up
docker-compose run api rake db:create
docker-compose run api rake db:migrate
```

## To test Front-End
```
cd contact-app
npm test
```

## To test Back-End
```
Add sudo if you need permissions :)
docker-compose run api rspec
```
